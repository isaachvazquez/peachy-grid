Nesting Columns

Breakpoints

Conditional Breakpoints

Conditional breakpoints allow you to change the column size at each specific breakpoint.

For example:

<div class="col-100 bp-l-50"></div>
<div class="col-100 bp-l-50"></div>

will create 2 columns that are 100% wide until the large breakpoint, where they will then both become 50% wide.

Utility Classes

No Gutter (.no-gutter)

The .no-gutter utility class removes the gutters from columns.

Hide at Breakpoint (.hide-breakpoint)

These are used for hiding specific columns at one or multiple breakpoints.

Setup & Usage

Peachy Config

The Peachy Defaults are as follows:

Gutters

  • side-gutter: 10px
  • top-gutter: 10px
  • bottom-gutter: 10px

Breakpoints

  • breakpoint-s: 400px
  • breakpoint-m: 640px
  • breakpoint-l: 960px
  • breakpoint-xl: 1024px
  • breakpoint-xxl: 1600px

If these work for you, you can just add peachy.css to your stylesheets, and reference it in your markup and you should be good to go.

Otherwise, if you would like to customize your gutters, and breakpoints, follow these steps:

Terminal

  • Update the variables in peachy-config.scss
  • If using Gulp, in the terminal, run gulp build
